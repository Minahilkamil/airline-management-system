# Airline Management System ✈️

![C++](https://img.shields.io/badge/C++-17-blue.svg)
![Qt](https://img.shields.io/badge/Qt-6.x-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

A robust desktop application built with **C++17** and **Qt 6** for comprehensive airline management. This system provides a seamless experience for both passengers and administrators, featuring smooth animations and a user-friendly interface.

## 🚀 Features

### 👤 User Module
- **Smart Registration & Login**: Secure authentication for passengers.
- **Flight Booking**: Search and book tickets to various destinations.
- **Ticket Review**: View and manage your booked flights.
- **Complaints System**: Lodge and track service-related complaints.
- **PakTour Packages**: Explore exclusive tour packages for travelers.

### 🔑 Admin Module
- **Secure Admin Portal**: Dedicated access for airline staff.
- **Ticket Management**: Cancel or update flight details and ticket statuses.
- **Complaint Resolution**: Review and address passenger feedback.
- **Data Monitoring**: Track active users and system logs.

## 🛠️ Technologies Used
- **Language**: C++17
- **Framework**: Qt 6 (Widgets & Core)
- **Animations**: Qt Property Animation for dynamic UI transitions.
- **Storage**: File-based persistence using flat text files (`.txt`).

## 🏁 Getting Started

### Prerequisites
- **Qt Framework**: Version 6.x (Recommended)
- **Compiler**: MSVC 2019+, GCC 9+, or Clang 10+
- **Qt Creator**: For the best development experience.

### Installation & Run
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/airline-management-system.git
   ```
2. **Open Project**: Launch Qt Creator and open `Airline/Airline.pro`.
3. **Configure**: Select your Qt Kit (e.g., Desktop Qt 6.7.x).
4. **Build & Run**: Press `Ctrl + R`.
